计算机网络基础及网络模拟器的初步认识（第一节）

初级名词解释

行业现状这里不再赘述，直接进入计算机网络基本名词的解释：

1.数字信号：由二进制比特流组成，简单来说就是一连串的0和1。

2.LAN（局域网）：是一种在特定区域内由多台计算机设备（如服务器、工作站）通过线缆（如同轴线缆、双绞线、光缆等）连接起来实现的资源共享的计算机网络
忽略上面这些繁琐的解释，目前我们只需要知道一切能上网的智能化终端设备（如📱、💻）都处于局域网内。常见的局域网相信会上网的人都耳熟能详，如家庭网、校园网、企业网、园区网等。

3.MAN（城域网）：是一种覆盖城市或大型城镇区域的计算机网络，通常用于连接城市内的不同地点，如企业、学校、医院等。简单来说，城域网就是用于连接一个城市内部不同局域网的网络。

4.WAN（广域网）；跨越地理位置的综合型网络，通常跨越不同国家甚至各大洲，使用公共网络（如互联网）或专用网络（如MPLS）连接不同的网络节点 目前世界上最大的广域网是Internet（因特网）。

在日常生活中，我们常说的上网冲浪指的就是在一个局域网内通过链接到广域网来获取到各个城域网乃至其他局域网内部的网络资源；上网上网，上的就是广域网。因此，我们可以简单理解为 ICT（Information and Communication Technology，信息与通信技术）网络技术所研究的就是广域网的接入技术。


图片来源：深入理解计算机网络（四）：网络层 | Taogen’s Blog

6.实现用户上网的条件
<1> 物理条件：传递信号的介质，如网线、光纤、电话线、电磁波（无线信号）
<2> 网络通信协议：TCP/IP（Transmission Control Protocol / Internet Protocol，传输控制协议/网际协议）协议族，详细介绍可以参考 TCP/IP介绍 | 菜鸟教程

7.IP 地址：在网络中用于唯一标识一台网络设备。由于 IPv6（Internet Protocol Version 6）尚未普及，现在我们常说的 IP 地址通常指 IPv4（Internet Protocol Version 4）地址。通常以三点分十进制数形式表示，如 192.168.1.1 。但计算机内部只能存储二进制数据，故其原型由32位二进制数组成 。如上面提到的192.168.1.1，可通过以下方法快速转换为二进制数：

8.IP 地址配置规则
从上面 IP 地址的二进制形式我们可以很容易得出，其十进制形式的取值范围在0 ~ 255之间。但在配置 IP 地址时，并不是每一个数值都可以使用。
在配置 IP 地址时，我们需要遵循以下规则：

<1> 首位的取值范围在1~223之间
<2> 中间两位的取值范围在0~255之间
<3> 末尾的取值范围在1~254之间

具体原因如下：

首先需要知道，末尾为0的 IP 表示网络的起始地址，通常会用于标识整个网络。在 IP 地址中，网络地址和主机地址是分开的，网络地址用于标识网络，而主机地址用于标识网络中的设备。为避免混淆，IP 地址末尾不为0便成为了一种约定俗成的做法；而末尾为255的 IP 地址则用于广播（允许数据从一个源发送到网络中的所有设备），被称为广播地址，末尾不为255同样是为了避免混淆不同标识功能的 IP 地址。

至于首位的范围，则需要引入IP 地址的分类来解释：

在 IPv4 中，我们以首位的取值范围为标准将 IP 地址分为 A、B、C、D、E 五大类：

A类地址：首位的范围在1~126之间
B类地址：首位的范围在128~191之间
C类地址：首位的范围在192~223之间

D类地址：首位的范围在224~239之间
E类地址：首位的范围在240~255之间

其中，D类地址用于组播（允许数据从一个源发送到多个接收者）、E类地址用于科研，这两类地址不能私下使用 。因此，在设置 IP 地址时，首位的范围在1~223之间（需要排除三个特殊地址段，详情可参考 被保留的5个特殊地址段）。

9.IP 地址的结构
针对 IP 地址本身，我们根据各部分标识功能的差异将其细分为网络位和主机位：

网络位：用于标识整个网络，是一个范围。如 192.168.10.100 的网络位是 192.168.10，表示网络地址 （注意，这里所提到的“网络地址”和上面提到的“用于标识整个网络的‘网络地址（起始地址）’”并不是同一个概念） 为192.168.10.0~192.168.1.255的所以网络设备组成的通信网络。这就是上面所提到的，IP 地址末尾不为0的原因。
主机位：网络中主机的标识，具体到一个网络中的一个用户。如上面的 IP 地址主机位为100 。
我们可以将网络位简单理解为一个 IP 地址的“姓”，而主机位就是它的“名”。

在不同种类的 IP 地址中，不同网络所拥有网络地址数量也不同。
例如：
对于A类地址 10.100.100.100 ，其网络位是 10.0.0.0 。因此，算上起始地址和广播地址，单个A类地址在网络中所拥有的地址数量就是256^3=2^24 。
同理，对于B类地址 172.16.100.100 ，其网络位是 172.16.0.0 ，则单个B类地址在网络中拥有256^2=2^16个网络地址 ；
单个C类地址在网络中拥有256^1=2^8个网络地址 。

是否感觉有一些混乱？计算机也怎么觉得
因此，为了准确标识一个 IP 地址所处的网络范围，我们引入子网掩码这个概念。
子网掩码通常与 IP 地址成对出现，在给网络设备配置 IP 地址时，后面必须配上子网掩码，否则 IP 无效 。

一个 IP 地址可以拥有不同的子网掩码，以标识它们处于不同的网络范围。
对于A、B、C三类地址，它们默认的子网掩码分别是 255.0.0.0、255.255.0.0、255.255.255.0。

那么子网掩码如何标识一个 IP 地址所处的网络范围呢？
将一个 IP 地址和其子网掩码做与运算，我们可以得到这个 IP 地址的网络号（注意其与网络位的区分）。
如 10.1.1.1 与 10.2.2.2，使用它们默认的子网掩码 255.0.0.0 ，分别对它们进行与运算：

另外，在配置网络信息的过程中，我们常常会看到类似于 192.168.1.1/24、172.10.12.100/16、10.1.1.1/8 等这样 IPv4地址+/+n 的地址格式，其中/ 后的数值 n 被称为前缀长度，表示表示子网掩码的前 n 位是 1，其余位是 0 。

10.三大基本网络设备：

路由器（Router）：连接广域网的设备
交换机（Switch）：连接局域网的设备，接口多，可使多台终端设备接入
调制解调器（Modren）：数字信号转换为模拟信号（调制）以便通过电话线传输，同时将接收到的模拟信号转换回数字信号（解调）。我们耳熟能详的光猫就是调制解调器的一种，全称为光网络调制解调器（Optical Network Modem，简称ONU），用于光信号和电子信号的相互转换

连接线路及 DHCP 配置（第二节）

名词解释

网线

连接各个网络设备的线路被称为网线，学名双绞线 ，由两根绝缘铜线紧密相绕组成，以减少电磁干扰（EMI）和射频干扰（RFI）。
我们将其分为直通线和交叉线

其中，从不同的角度来看，这两类双绞线有以下特点：

1.根据连接对象的类型：
直通线：在模拟器上体现为一条直实线，用于连接不同种类的网络设备。如路由器-交换机、交换机-PC等。
交叉线：在模拟器上体现为一条直虚线，拥有连接相同种类的网络设备。如路由器-路由器、交换机-交换机、路由器-PC（这二者均具有网络适配器，被划分为相同种类）等。

2.根据网线两端线缆的排布顺序：
线序1：白橙 橙 白绿 蓝 白蓝 绿 白棕 棕
线序2：白绿 绿 白橙 蓝 白蓝 橙 白棕 棕
直通线：1-1 或 2-2
交叉线：1-2

注意：一台路由器上的不同接口不能配置相同的网段 。而处于不同网段的网络设备又不能直接通信，这时我们就需要一个能够实现跨网段通信的介质，于是，网关应运而生。

网关，网络中一个非常重要的概念，终端设备联网四大基本参数之一，通常指的是网络中的一个设备或软件，用于连接不同的网络或网络协议。主要功能是作为两个网络之间的桥梁，实现数据的传输和转发。我们可以将其简单理解为网络通信界的中转站（路由器网关）或海关（防火墙网关等）。

网关的本质是一个已存在的 IP 地址，在配置过程中，我们通常将距离 PC 最近的路由器接口的 IP 地址配置为本台 PC 的默认网关 。

DHCP 模拟配置

DHCP（Dynamic Host Configuration Protocol）是一种网络协议，用于自动分配IP地址和其他网络配置参数给网络设备，如计算机、打印机等。DHCP服务器负责管理IP地址池，并在设备首次连接到网络时，自动为设备分配IP地址和其他网络参数。

终端设备联网四大基本参数

配置前我们需要首先了解以下参数：
IP 地址（ip address）：标识网络设备身份，用于路由和定位数据包，确保数据能够正确地传输到目标设备。

子网掩码（Subnet Mask）：用于将IP地址划分为网络部分和主机部分，标识网络设备所处的网络范围，帮助路由器确定哪些设备属于同一个子网，从而决定是否转发数据包。

默认网关（Default Gateway）：网络中的一个设备或路由器，用于连接不同的网络或互联网，实现跨网段通信 。

域名服务器（DNS Server）：用于将域名（如www.example.com）转换为IP地址，将域名和IP地址相互映射，使得用户可以通过易于记忆的域名访问互联网上的资源，而不是记住复杂的IP地址。DNS只影响上网方式不影响上网结果 。
